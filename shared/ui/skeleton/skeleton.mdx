# Skeleton

O **Skeleton** é um componente de interface utilizado dentro do design system para compor interfaces consistentes.

## Como funciona

Este componente encapsula estilos e comportamento reutilizável.

## Estrutura básica

```tsx
import { Skeleton } from "@/components/ui/skeleton"

export function Example() {
  return (
    <Skeleton>
      Conteúdo
    </Skeleton>
  )
}
```
