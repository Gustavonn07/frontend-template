# Tooltip

O **Tooltip** é um componente de interface utilizado dentro do design system para compor interfaces consistentes.

## Como funciona

Este componente encapsula estilos e comportamento reutilizável.

## Estrutura básica

```tsx
import { Tooltip, TooltipContent, TooltipTrigger } from "@/components/ui/tooltip"

export function Example() {
  return (
    <Tooltip>
      <TooltipTrigger>Trigger</TooltipTrigger>
      <TooltipContent>Conteúdo</TooltipContent>
    </Tooltip>
  )
}
```
