# ToggleGroup

O **ToggleGroup** é um componente de interface utilizado dentro do design system para compor interfaces consistentes.

## Como funciona

Este componente encapsula estilos e comportamento reutilizável.

## Estrutura básica

```tsx

export function Example() {
  return (
    <ToggleGroup>
      Conteúdo
    </ToggleGroup>
  )
}
```
