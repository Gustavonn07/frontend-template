# Button

O **Button** é um componente de interface utilizado dentro do design system para compor interfaces consistentes.

## Como funciona

Este componente encapsula estilos e comportamento reutilizável.

## Estrutura básica

```tsx
import { Button } from "@/components/ui/button"

export function Example() {
  return (
    <Button>
      Conteúdo
    </Button>
  )
}
```
